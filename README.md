# Wagner's dotfiles

It's under construction, works only on macOS.

## Instalation
clone this repo on your `$HOME` directory and
```
cd ~/dotfiles
sh init.sh
```

### Actual features

- homebrew and cask
- nvm and node
- yarn
- hub
- zsh and oh-my-zsh
- tmux
- z
- GNU terminal commands updated
- usefull alias and bash functions
